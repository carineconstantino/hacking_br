## 👾 HTML Scan [Web app]
### 🇧🇷
<p>Usando Python3 e o Streamlit, o HTML Scan faz uma análise no código HTML e Javascript em busca de: 
- links, 
- diretórios, 
- credenciais e chaves de API, 
- cookies e o local storage, 
- campos de formulário, 
- domínios, 
- endereço IP.
