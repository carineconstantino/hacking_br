## 👾 NMAP [Ultimate Guide]
### 🇧🇷
Simple Scan
```
nmap -Pn -sS -sV -sC [IP] -oN [file].txt
```
- -Pn = disable ping
- -sS = syn scan

### 🇺🇸
Zoltar tool make a domain enumeration from a IP address by searching a reverse DNS registers.


## ⏩ Exemplo
```
python3 zoltar-dns-reverse-search.py -ip [IP]
```
