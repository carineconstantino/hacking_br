## 👾 Zodar
### 🇧🇷
Zodar é uma ferramenta em Python3, que faz uma enumeração de subdomínios usando a resolução DNS.

### 🇺🇸
Zodar tool make a subdomain enumeration by DNS resolve.


## ⏩ Exemplo
```
python3 zodar.py -domain [domain]
```
